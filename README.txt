Pagina web responsiva diseñada aplicando conceptos de mobile first y la metodología BEM.

Realizada para practicar los conceptos aprendidos hasta ese momento. César Mongez

Vamos a crear un sitio web para “shopTravel”, donde permita mostrar la información de la
empresa, se le solicitará:
- Construir estructura HTML utilizando etiquetas semánticas, mínimo 3, como
header, section, footer, nav o article.
- Crear un archivo Css.
- Agregar imágenes al proyecto.
- La página debe tener títulos y subtítulos para las secciones.
- La página debe tener una barra de navegación fija en el borde superior que ocupe
el 100% del ancho.
- Todas las páginas deben contener un footer que incluya íconos de redes sociales.
- Realizar un banner, que será una imagen que ocupe el 100% del ancho como una
imagen de fondo y sobre la imagen colocar : “ Bienvenidos a shopTravel “
- Vamos a tener una section de productos. Se deberá ver con una imagen y una
descripción, seran 3 productos distribuidos horizontalmente , un producto al lado
del otro.
